# MAM-SORT
基于监控视频的车辆跟踪<br>
方法：<br>
基于检测的跟踪，车辆检测使用YOLOv7；重识别网络使用CA-OSNet；关联匹配算法使用运动特征优为主，外观特征为辅的关联匹配算法（MAM）。<br>
![](https://github.com/wuchuimao/MAM-SORT/raw/main/images/MVI-39311.gif)<br>
# 数据集
## 车辆检测与跟踪数据集
车辆检测与跟踪的数据集使用UA-DETRAC数据集，该数据集分为训练集60个视频的82085张图片，测试集40个视频的56167张图片。视频以每秒 25 帧 (fps) 的速度录制，分辨率为 960×540 像素。车辆种类分为轿车、公共汽车、面包车和其他。
